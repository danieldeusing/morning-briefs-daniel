**fit_reason** (1 Satz) belegt den Score mit der Komponenten-Aufschlüsselung, z. B. „stack 43 + loc 20 + rate 12 + intrigue 5 + track 10 = 95".

- **Wenn die Sektion leer wäre** (kein passender Treffer heute): die ganze
  Sektion mit einer einzigen Zeile „Heute keine passende TED-Ausschreibung im
  Filter" weglassen, nicht auffüllen.

## Markt-Signal (no coaching)
Wiederkehrende Skills in den Listings nüchtern notieren (z. B. „heute 5× Angular ≥17 in twiceD-Listings"). **Keine** „CV-Update könnte sich lohnen"-Sprache.

## Spur 3 — EU-Projektausschreibungen (TED)
- **TED Search API** (kein API-Key nötig): `https://api.ted.europa.eu/v3/notices/search`
- **Swagger / Schema**: `https://api.ted.europa.eu/swagger` — bei Unklarheit über
  Query-Syntax oder Response-Felder dort die aktuelle Definition lesen, nicht raten.
- **CPV-Codes** (primärer Filter): `72200000` (Software-Programmierung +
  Beratung), `72240000` (Systemanalyse + Programmierung), `72500000` (Computer-
  Services), `72220000` (Systemberatung), `72260000` (Software-bezogene Services).
  Mit Prefix-Match `72*` lassen sich alle erfassen. Adjazent zulässig: `48*`
  (Software-Pakete). **Ausschluss-CPV**: `73*` (R&D), `30*` (Hardware), reine
  Telekom (`64*`).
- **Länder-Filter**: `DEU` (primär), `AUT` (sekundär), EU-Institutionen
  (BE/LU für Brüssel/Luxemburg-Sitz) wenn Remote-fähig. CH nicht in TED.
- **Status / Frist**: nur aktive Ausschreibungen, Abgabefrist ≥ heute + 5
  Werktage.
- **Notice-Detail-URL** auf TED.europa.eu: `https://ted.europa.eu/de/notice/-/detail/<id>`
  (für DE-Anzeige; alternativ `/en/`).
- **Volltext / eForm-XML** (für Pflicht-Kriterien wie Mindest-Umsatz, FTE-
  Anforderung, Referenz-Anzahl, Zertifikate): von der Detail-Seite den XML-/
  eForm-Link folgen oder per API einzelnen Notice fetchen — **aber nur für die
  ~5–10 Kandidaten, die Stack-Match haben** (Fetch-Ökonomie). Nicht jeden Such-
  Treffer voll laden.
- **Rate-Limit**: API ist öffentlich, aber zurückhaltend bleiben — eine Search-
  Anfrage pro Lauf + ggf. ~10 Detail-Fetches. Kein Polling.
