Anthropic-Billing-Split: Agent-SDK bekommt eigenen Topf [1]

Heute, 15. Juni, tritt die wichtigste Änderung des Quartals für Daniels eigene Tool-Kette in Kraft: programmatische Claude-Nutzung wird vom Abo-Limit abgekoppelt und zieht aus einem separaten monatlichen Credit-Pool. Angekündigt wurde das am 14. Mai, der Stichtag ist jetzt.[1]

Credit-Topf vs. Abo-Limit: Bisher hatte ein Max-Abo ein einziges „Wasserglas" pro 5 Stunden, aus dem alles trank — der Mensch am TUI genauso wie ein Background-Skript via claude -p. Wenn ein lang laufender Agent über Nacht den Glas leergesoffen hat, war morgens auch das interaktive Arbeiten rate-limited. Ab heute gibt's zwei Gläser: das alte Abo-Glas für interaktive Arbeit (TUI, Cowork, claude.ai) und ein separates, in Dollar gemessenes Glas für alles Programmatische. Wer mit Skripten arbeitet, blockiert sich nicht mehr selbst — aber der Skript-Topf ist endlich, läuft nicht über in den nächsten Monat, und ist nicht im Team poolbar.

Was in den neuen Topf wandert: Claude Agent SDK, claude -p (headless CLI), Claude-Code-GitHub-Actions, Third-Party-Apps via Agent Client Protocol.[1] Was am Abo-Limit bleibt: interaktive Claude Code (Terminal-TUI, IDE-Plugin), Claude Cowork, claude.ai (Web/Desktop/Mobile).[1]

Fable 5 + Mythos 5 weltweit abgeschaltet — Export-Control [2][3][4]

Anthropics neue Mythos-Klasse — Fable 5 (GA seit 09.06.) und das Preview Mythos 5 — sind seit 12. Juni 2026, 17:21 ET weltweit nicht mehr nutzbar.[3] US-Commerce-Secretary Howard Lutnick hat per Brief an Dario Amodei eine Export-Control-Direktive erlassen, die den Zugriff für jeden Foreign National sperrt — innerhalb und außerhalb der USA, einschließlich Anthropics eigener internationaler Mitarbeiter.[3] Anthropic hat den Zugriff für alle Nutzer abgedreht, weil sich der Foreign-National-Status in einer öffentlichen API nicht zuverlässig prüfen lässt.

Im GitHub Copilot wurde Fable 5 am Freitag 12.06. taggleich gesperrt: „effective today, access to Claude Fable 5 has been suspended across all GitHub Copilot experiences"[2] — drei Tage nach GA-Start am 09.06.[2]

Export Control auf ein API-Modell: Klassischerweise regelt das US-Handelsministerium den Export von Chips, Crypto-Algorithmen oder Dual-Use-Software. Hier wird zum ersten Mal ein generatives KI-Modell wie Dual-Use-Tech behandelt — Vorwurf: die Fähigkeit, „Schwachstellen in Code zu finden", sei für Cyber-Angriffe weiterverwendbar. Anthropics Gegenposition: dieselbe Fähigkeit hat auch GPT-5.5; „dieser Maßstab würde branchenweite Neuveröffentlichungen stoppen."[3]

Trigger laut Marktechpost: ein „narrow and non-universal" Jailbreak, den ein „highly credible, trusted partner" Anthropics und der US-Regierung demonstriert hat — die Technik fragt das Modell, eine Codebasis auf Sicherheitslücken zu durchsuchen.[3][4]

Keine offizielle Aufhebungs-Frist — bis dahin ist Opus 4.8 das beste verfügbare Anthropic-Modell (88,6 % SWE-Bench Verified, siehe nächste Sektion).

Gemini CLI → Antigravity CLI: Donnerstag ist Stichtag [6]

Google macht am 18. Juni 2026, 23:59 PT (Fr 19.06., 08:59 MESZ / 03:59 BRT) das Gemini CLI zu.[6] Ersatz ist das seit dem 19. Mai allgemein verfügbare Antigravity CLI — Go-basiert (statt Python), asynchrone Multi-Agent-Workflows ohne Terminal-Lock, vereinheitlicht mit dem Antigravity-2.0-Desktop-Harness.[6]

Wechsel diese Woche, nicht in vier Wochen. Wer Gemini CLI in CI- oder lokalen Skripten gepinned hat, fliegt ab Freitag mit „command not found" raus. Ausnahme: Gemini-Code-Assist-Standard/Enterprise-Lizenzkunden behalten ihren Zugang — EOL trifft nur Free-/Login-Nutzer.[6]

Multi-Agent-Workflow (CLI): Ein einzelner Agent macht in Schleife: Plan → Tool-Call → Ergebnis → nächster Plan. Bei langen Aufgaben (indizieren, Tests laufen, Patch schreiben) blockiert das das Terminal. Multi-Agent: das Hauptwerkzeug delegiert an Sub-Agenten, die parallel im Hintergrund tickern, und liefert das Ergebnis zurück. Im Antigravity-CLI nativ; im alten Gemini CLI musstest du das selbst nachbauen.

GitHub Copilot: Usage-Based-Billing live, neue Max-Stufe [7]

Seit dem 1. Juni 2026 rechnet GitHub Copilot Premium-Anfragen in AI Credits ab: 1 AI Credit = 0,01 USD.[7] Code-Completions und Next-Edit-Suggestions bleiben frei (nicht in Credits abgerechnet), das Limit greift erst bei Chat, Agent-Mode, Pull-Request-Reviews und Spaces.[7]

Die Max-Stufe ist neu — für „sustained, high-volume agent workflows": $100 Festpreis, $200 Credit-Budget, faktisch Volumen-Verdopplung pro Dollar. GitHubs Begründung: „a quick chat question and a multi-hour autonomous coding session can cost the user the same amount … the current premium request model is no longer sustainable."[7] Visual Studio Magazine titelte am 27.04.: „You Will Get Less, but Pay the Same Price" — Power-User sehen kein Volumen-Plus, sondern höhere Effektiv­kosten pro Run.[7]

Parallel zu Anthropic Auffallend: GitHub und Anthropic ziehen im Gleichschritt — beide trennen Background-Verbrauch vom interaktiven Limit und stellen ihn auf $-Credits um. Agenten-Workloads explodieren im Token-Verbrauch; ein einheitliches Abo-Limit ist anbieter­seitig nicht mehr tragbar. Für Daniel: Verbrauchs-Monitoring ist ab sofort Tool-Hygiene.

OpenCode überholt Claude Code bei den Stars [9]

Der bemerkenswerteste OSS-Datenpunkt der Woche: OpenCode — MIT-lizenzierter Terminal-Coding-Agent — steht auf 172.198 GitHub-Stars und ist damit der meist-gestarrte Open-Source-Coding-Agent.[9] Zum Vergleich: Claude Code ~124.000, Gemini CLI ~105.000, openai/codex ~90.000.[9] OpenCode unterstützt nach byteiota-Quelle 75+ AI-Provider (Claude, GPT, Gemini, Ollama) — die Modell-Agnostik ist der zentrale Verkaufsgrund.

Reality-Check Stars ≠ Qualität. Ein medium.com-Reviewer hat OpenCode auf identischen Tasks gemessen: ~78 % langsamer als Claude Code.[9] Einzelstimme, kein Benchmark — deckt sich aber mit dem Muster, dass „75+ providers" oft eine Generic-Adapter-Schicht heißt. try-this auf ein eigenes Pet-Projekt loslassen — der Geschwindigkeits-Trade-off ist schnell spürbar; ist er irrelevant, ist Modell-Wahlfreiheit ein realer Vorteil.

Open-Weights: MiniMax M3 und Qwen3-Coder-Next [11]

Die zwei Welten nähern sich an, aber sie sind nicht konvergiert.
