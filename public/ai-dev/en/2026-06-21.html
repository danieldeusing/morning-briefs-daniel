Export control locks Fable 5 + Mythos 5 — ongoing [1][2][3]

The week's most important news is nine days old and still true: Anthropic's two top coding models, Fable 5 and Mythos 5, have been disabled since June 12 for all customers. The basis is an export-control order from the US Department of Commerce (Bureau of Industry and Security, signed by Commerce Secretary Howard Lutnick) that prohibits access by "any foreign national" — a restriction that in practice could only be enforced by complete shutdown.[1]

Export-control order: Imagine the government tells a bakery it can't sell a specific bread to foreigners — but the bakery can't check who's foreign at the checkout. So it takes the bread off the shelf entirely. That's exactly what Anthropic had to do with Fable 5 and Mythos 5 instead of just blocking individual users.

The official justification: a jailbreak risk in cybersecurity tasks. Anthropic disputes the severity — they reviewed the technology and found "a small number of already-known, minor vulnerabilities."[1] Anthropic met with US officials on June 15–16, and a return date has not yet been confirmed.[2] A circulating rumor of "return around July 1" is not confirmed by Anthropic — we deliberately don't cite it as fact here.

All other models (Opus-4.8, Sonnet, and Haiku lines) continue to run normally.[1] In practice, this means: if you ran Claude Code with Fable 5 as the default model, you now fall back to Opus 4.8 — and that's noticeably weaker on Terminal-Bench (more on this in the Bench section). Take For tool choice, this is a hard argument for a fallbackModel chain and provider diversity; a single model can disappear overnight by government order.

Gemini CLI is dead for consumers — agy takes over [4][5]

Google's Gemini CLI and Gemini Code Assist have been disabled for consumer accounts since June 18 — affecting Free, Google AI Pro, and Ultra tiers; Enterprise keeps access.[4] The successor is Antigravity CLI, a Go-based, closed-source tool called agy.[5]

It's more than a rebrand: the old Gemini CLI was open, the new one is not. Google itself admits Antigravity "has no 1:1 feature parity from the start," but it will offer agent skills, hooks, subagents, and extensions.[4] For GitHub integrations, a deprecation window runs out with full shutdown on July 17 — worth migrating pipelines before then.

Heads up CI/CD pipelines that call gemini as a build step break now. If you've wired Gemini CLI into actions or scripts, check today, not July 16.

On Terminal-Bench 2.1, Gemini CLI last scored 70.7% (Gemini 3.1 Pro, entry from 05/05) — solid, but never top-tier. An Antigravity CLI submission doesn't yet exist on the leaderboard.[4]

MCP Ecosystem [10][11]

The AWS MCP Server is generally available (GA) — a managed remote MCP server that gives coding agents authenticated access to AWS services.[10]

MCP (Model Context Protocol): Like a USB standard for AI tools — instead of building a separate AWS integration for each agent, everyone speaks the same protocol, and one MCP server provides the connection.

The standard itself is advancing in the background: the 2026-07-28-RC brings a stateless protocol core (no more initialization handshake), formal extensions, and MCP Apps (server-rendered HTML in sandboxed iframes).[11] The PulseMCP directory continues growing by roughly 1,000 indexed servers per month — Q1–Q2 2026 combined estimated to exceed 16,500 entries.[11]

Terminal-Bench 2.1: Peak without a bookable champion [6][7]

On the primary Terminal-Bench-2.1 table (tbench.ai, 13 entries, latest from 06/17), Codex CLI + GPT-5.5 leads narrowly at 83.4% over Claude Code + Fable 5 at 83.1%.[6] The irony: the model in second place is exactly the one locked down by regulation this week. Start Claude Code today and you won't get that 83.1% — the next available Claude Code configuration is Opus 4.8 at 78.9%.

Terminal-Bench: A test where an AI agent solves real tasks at a command line — building files, turning green tests, fixing setups. You measure the pair of harness (the tool, e.g. Claude Code) and model (the brain, e.g. Fable 5), not the model alone.
Terminal-Bench 2.1 — Harness + Model, as of 06/21/2026 (entries through 06/17)

On the toughest tasks, the picture shifts: Terminal-Bench Hard has Fable 5 leading at 62.9% over GPT-5.5 (xhigh) at 60.6%.[7] The gap is real — and makes the lockout all the more frustrating for demanding agent work.

Read skeptically: Terminal-Bench scores carry wide confidence intervals (±2–3 points), and 83.4% vs. 83.1% is statistically a tie, not a "winner." The top-tier race is tighter than the table suggests.

It's Sunday, June 21, and today's AI coding brief is shaped not by releases, but by two regulatory decisions that pull tools off the shelf.

The main story is nine days old and still true. Anthropic's two strongest coding models, Fable five and Mythos five, have been locked down since June 12 for all customers. The reason is an export-control order from the US Department of Commerce that forbids access by foreign nationals. And because Anthropic can't verify citizenship at checkout, both models had to come offline completely. There's still no confirmed return date. If you were running Claude Code with Fable five, you're falling back to the weaker Opus four point eight.

That's why it stings so much. The same models are leading the benchmarks. On Terminal-Bench two point one, Codex with GPT five point five is slightly ahead at eighty-three point four percent, with Claude Code using Fable five just behind at eighty-three point one. On the hardest tasks, Fable five is even in the lead. So the best available tool is currently without its best model. Though you should read these numbers skeptically — the measurement uncertainty is bigger than the spread at the top.

The second shutdown comes from Google. The Gemini command line has been dead for consumer accounts since June 18. Its replacement is called Antigravity and uses the agy command, and unlike the old tool, it's no longer open source. If you've wired the old Gemini CLI into your build pipelines, you should fix that this week, because the GitHub integration will be completely shut down on July 17.

Beyond that, there were real releases. Cursor version three point eight brings automations — agents that run continuously and can be spun up directly from your session, with Slack and GitHub triggers. Claude Code in recent versions blocks destructive Git commands, which is a real security gain especially for unattended runs. And the open harness OpenHands ships version one point eight with LLM profiles and sub-agent delegation — an open alternative just as two commercial options vanish.

What to watch next week: every word from Anthropic about Fable and Mythos returning — and whether a first Antigravity CLI submission appears on the Terminal-Bench leaderboard. Until then: plan around availability, not features.
