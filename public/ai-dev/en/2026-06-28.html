GPT-5.6: Sol, Terra, Luna — big ambitions, one missing number

OpenAI unveiled the GPT-5.6 family on June 26, tiered into three levels: Sol (flagship), Terra (production/balance), and Luna (fast/cheap). It's a Restricted Preview for only ~20 pre-approved partners via API and Codex — nothing Daniel can test himself today. OpenAI "shared the models and plans first with the US government" before broader rollout — a pattern shaping this week.[1]

Reasoning modes (max vs. ultra): A "reasoning mode" is how the model thinks before answering. max is like a person pulling a problem through one long, deep thought. ultra is like a small team working in parallel on pieces and coordinating. GPT-5.6 offers both — that's why "Sol (ultra)" and "Sol (max)" appear separately in benchmarks.[1]

On Terminal-Bench 2.1 — the agent benchmark measuring whether a model can solve real tasks in a shell — OpenAI reports Sol at 91.91 % (ultra) or 88.76 % (max). Terra lands at 84.3 %, equaling Fable 5 before its lockdown. Sol also hits 50.9 % on "Agent's Last Exam" in code mode — the only model past the halfway mark.[1][2]

Heads up Two caveats that belong together. First: these scores come from OpenAI's own announcement — the official tbench.ai board (as of 06-17) doesn't list GPT-5.6 Sol yet, so they're not independently measured. Second: OpenAI has published no SWE-Bench score (neither Verified nor Pro) at preview launch — the number Daniel normally relies on is simply missing. Until it arrives, GPT-5.6 is a blank spot in his primary comparison dimension, not a winner.

Pricing: Sol costs $5 input / $30 output per 1M tokens, Terra $2.50/$15, Luna $1/$6. That's roughly Opus-4.8 class for the flagship — but without bookable access and no comparable SWE-Bench score, it's a claim, not a tool yet.[1]

For local/agentic harnesses where token budgets hit your wallet directly, that's a more relevant move than the next benchmark percent.[9]

Fable 5 remains dark — Opus 4.8 is the active top model

The export-control saga dominates the entire frontier field. Since the US order on June 12, Fable 5 and Mythos 5 have been blocked to non-US users. On June 27, Mythos 5 was partially restored — but only for Annex-A entities, their foreign staff, US civil agencies, and national labs; everyone else needs an export license. Fable 5 remains completely offline as of June 28, shows zero traffic for two weeks, and has no restoration date.[3][4]

What that means in practice The world's two best coding models (Fable 5: 95.0 % SWE-Bench Verified, Mythos Preview: 93.9 %) are off-limits to Daniel. Anthropic explicitly confirmed: "Access to all other Anthropic models is not affected." That makes Opus 4.8 the de facto frontier fallback — and that's no consolation prize: 88.6 % SWE-Bench Verified and 69.2 % SWE-Bench Pro, nearly 5 points ahead of Opus 4.7 and over 10 ahead of GPT-5.5 on Pro.[5][6]

Anthropic's stated trigger: a "narrow potential jailbreak" consisting of letting the model read a codebase and fix software bugs — a capability Anthropic itself classes as "broadly available in other models." That framing gained weight today: Chinese lab Zhipu's GLM 5.2 has reached Mythos' vulnerability-detection abilities at benchmark level — undercutting the containment logic behind the ban.[4]

ARD Spec: a phone book for agent tools

On June 17, eleven companies — Cisco, Databricks, GitHub, GoDaddy, Google, Hugging Face, Microsoft, NVIDIA, Salesforce, ServiceNow, Snowflake — published the Agentic Resource Discovery (ARD) spec (Apache 2.0, based on Linux Foundation AI Catalog working group).[10]

ARD vs. MCP: MCP is the plug that lets an agent hook into an external tool (a database, an API). The problem: until now you had to pre-register every MCP server in config. ARD is the phone book before it — an agent can look up at runtime "which MCP server can help with this task?" instead of everything being pre-installed. ARD doesn't replace MCP; it sits one layer above.[11][12]

Technically two pieces: a static ai-catalog.json under a well-known path describing an provider's MCP servers, agents, and OpenAPI tools — and a Registry API with POST /search that takes natural-language queries and returns ranked results. GitHub's "agent finder" already lets Copilot discover the right MCP server at runtime. Hugging Face has a reference tool (Discover) live.[11][12]

Why this matters: if it sticks, manual MCP-server registration goes away — agents find tools themselves. For anyone building agentic harnesses, that's the structurally most important announcement this week, even if it came quietly.

It's Sunday, June 28, and the frontier field in AI coding has just been reshuffled. Two big things define the week: OpenAI has launched GPT-5.6, and Claude's best model remains switched off.

OpenAI unveiled GPT-5.6 on Friday — in three tiers named Sol, Terra, and Luna. Sol is the flagship. But it's a tightly limited preview for only about twenty pre-approved partners. Nobody else can access it. OpenAI says Sol leads Terminal-Bench with just under ninety-two percent — but that's OpenAI's own number, not independently measured. More importantly: OpenAI has published no SWE-Bench score at all, which is exactly the number Daniel normally relies on. Until that comes, GPT-5.6 is simply a blank space in the most important comparison dimension.

The second big story is the export-control saga around Anthropic. Since June 12, the world's two best coding models, Fable Five and Mythos Five, have been blocked to non-US users. On Saturday, Mythos Five was partially restored, but only to US government agencies and a narrow set of entities. Fable Five is still completely offline. That means in practice: if you need a bookable frontier model for coding today, you land on Opus Four-Eight. And that's no consolation prize — Opus Four-Eight scores just under eighty-nine percent on SWE-Bench, making it the strongest actively available model.

Remarkably: today the Chinese lab Zhipu reported achieving Mythos' security capabilities at benchmark level. That undermines the exact logic used to justify the ban.

Quieter, but perhaps most important structurally: eleven companies, including Google, Microsoft, GitHub, and NVIDIA, published a specification called ARD. It's a kind of phone book for agent tools — an agent can look up at runtime which MCP server will help with a given task, instead of everything needing to be pre-installed. Anyone building agentic tools should keep an eye on this.

Among the tools themselves, Claude Code had the busiest week, with nested sub-agents, significantly less CPU load during streaming, and a safer Auto-Mode. Cursor and OpenHands also made moves. Aider, by contrast, is stalled — no new release since August last year.

One warning about the numbers: of one hundred entries on the SWE-Bench board, only a single one carries an independent verification seal. The other ninety-nine were submitted by the vendors themselves. Read each percentage accordingly.

What to watch next week: the first independent SWE-Bench scores for GPT-5.6 — and whether Gemini Three-Point-Five Pro actually hits its new July target.
