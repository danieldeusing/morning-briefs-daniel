La mejor herramienta, sin su mejor modelo

Domingo, 21 de junio — y el Día de Herramientas está a la sombra de dos decisiones regulatorias, no de dos lanzamientos. Los modelos Fable 5 y Mythos 5 de Anthropic están bloqueados desde el 12 de junio por orden de control de exportaciones de EE.UU. para todos los clientes — exactamente los modelos que ponen a Claude Code en la cima del Terminal-Bench. Y la Gemini CLI de Google está muerta para consumidores desde el 18 de junio, reemplazada por una herramienta de código cerrado. Si trabajas de forma agentiva, planeas esta semana alrededor de la disponibilidad, no de características.

  • Herramienta Cursor 3.8: Automatizaciones con agentes persistentes, disparador de emoji de Slack, 5 nuevos disparadores de GitHub.
  • Claude Claude Code 2.1.183–185: bloquea comandos Git destructivos, /config key=value, anidamiento de subagentes hasta 5 niveles.
  • Herramienta GitHub Copilot: desde 01/06 facturación de uso a través de AI Credits (1 Credit = $0,01); las completaciones de código siguen siendo ilimitadas.
  • OSS OpenHands v1.8.0: perfiles de LLM, agrupamiento de sandbox, delegación de subagentes.
  • MCP Ecosistema: AWS MCP Server GA, Core sin estado en 2026-07-28-RC, PulseMCP creciendo ~1.000 servidores/mes.

Control de exportación bloquea Fable 5 + Mythos 5 — en curso [1][2][3]

La noticia más importante de la semana tiene nueve días y sigue siendo verdadera: los dos modelos de código principales de Anthropic, Fable 5 y Mythos 5, han sido deshabilitados desde el 12 de junio para todos los clientes. La base es una orden de control de exportación del Departamento de Comercio de EE.UU. (Oficina de Industria y Seguridad, firmada por la Secretaria de Comercio Howard Lutnick) que prohíbe el acceso por "cualquier ciudadano extranjero" — una restricción que en la práctica solo podría hacerse cumplir mediante un apagón completo.[1]

Orden de control de exportación: Imagina que el gobierno le dice a una panadería que no puede vender un pan específico a extranjeros — pero la panadería no puede verificar quién es extranjero en la caja. Entonces retira el pan completamente de la repisa. Eso es exactamente lo que Anthropic tuvo que hacer con Fable 5 y Mythos 5 en lugar de solo bloquear usuarios individuales.

La justificación oficial: un riesgo de jailbreak en tarefas de ciberseguridad. Anthropic cuestiona la gravedad — revisaron la tecnología y encontraron "un pequeño número de vulnerabilidades ya conocidas y menores".[1] Anthropic se reunió con funcionarios de EE.UU. el 15–16 de junio, y una fecha de retorno aún no ha sido confirmada.[2] Un rumor circulante sobre "retorno alrededor del 1 de julio" no es confirmado por Anthropic — deliberadamente no lo citamos como hecho aquí.

Todos los otros modelos (líneas Opus-4.8, Sonnet y Haiku) continúan funcionando normalmente.[1] En la práctica, esto significa: si ejecutabas Claude Code con Fable 5 como modelo predeterminado, ahora te repliegas a Opus 4.8 — y es notablemente más débil en Terminal-Bench (más sobre esto en la sección Bench). Conclusión Para la elección de herramienta, este es un argumento fuerte para una cadena fallbackModel y diversidad de proveedores; un único modelo puede desaparecer de la noche a la mañana por orden gubernamental.

Gemini CLI está muerta para consumidores — agy toma el relevo [4][5]

La Gemini CLI de Google y Gemini Code Assist han sido deshabilitadas para cuentas de consumidores desde el 18 de junio — afectando planes Free, Google AI Pro y Ultra; Enterprise mantiene acceso.[4] El sucesor es la Antigravity CLI, una herramienta de código cerrado basada en Go llamada agy.[5]

Es más que un cambio de marca: la antigua Gemini CLI era abierta, la nueva no lo es. El propio Google admite que Antigravity "no tiene paridad de características 1:1 desde el inicio," pero ofrecerá habilidades de agentes, hooks, subagentes y extensiones.[4] Para integraciones de GitHub, la ventana de depreciación termina con apagón completo el 17 de julio — vale la pena migrar pipelines antes de entonces.

Atención Los pipelines de CI/CD que llaman gemini como paso de construcción se rompen ahora. Si has conectado Gemini CLI en acciones o scripts, verifica hoy, no el 16 de julio.

En Terminal-Bench 2.1, Gemini CLI puntuó 70,7% por último (Gemini 3.1 Pro, entrada del 05/05) — sólido, pero nunca de nivel superior. Una presentación de Antigravity CLI aún no existe en la tabla de clasificación.[4]

Ecosistema MCP [10][11]

El AWS MCP Server está disponible en general (GA) — un servidor MCP remoto administrado que proporciona a los agentes de código acceso autenticado a los servicios de AWS.[10]

MCP (Model Context Protocol): Como un estándar USB para herramientas de IA — en lugar de construir una integración de AWS separada para cada agente, todos hablan el mismo protocolo y un servidor MCP proporciona la conexión.

El estándar en sí avanza en segundo plano: el 2026-07-28-RC trae un protocolo sin estado (sin más apretón de manos de inicialización), extensiones formales y aplicaciones MCP (HTML renderizado por servidor en iframes en sandbox).[11] El directorio PulseMCP continúa creciendo aproximadamente 1.000 servidores indexados por mes — Q1–Q2 2026 combinados se estima que superan 16.500 entradas.[11]

Terminal-Bench 2.1: Pico sin campeón reservable [6][7]

En la tabla primaria Terminal-Bench-2.1 (tbench.ai, 13 entradas, última del 17/06), Codex CLI + GPT-5.5 lidera estrechamente con 83,4% sobre Claude Code + Fable 5 con 83,1%.[6] La ironía: el modelo en segundo lugar es exactamente el que fue bloqueado por regulación esta semana. Inicia Claude Code hoy y no obtendrás ese 83,1% — la siguiente configuración disponible de Claude Code es Opus 4.8 con 78,9%.

Terminal-Bench: Una prueba donde un agente de IA resuelve tareas reales en una línea de comandos — construyendo archivos, dejando pruebas verdes, arreglando configuraciones. Mides el par de herramienta (la herramienta, por ejemplo, Claude Code) y modelo (el cerebro, por ejemplo, Fable 5), no solo el modelo.
Terminal-Bench 2.1 — Herramienta + Modelo, a partir del 21/06/2026 (entradas hasta 17/06)

En las tareas más difíciles, la imagen cambia: Terminal-Bench Hard tiene Fable 5 liderando con 62,9% sobre GPT-5.5 (xhigh) con 60,6%.[7] La brecha es real — y hace el bloqueo aún más frustrante para el trabajo agentivo exigente.

Lee con escepticismo: las puntuaciones de Terminal-Bench llevan intervalos de confianza amplios (±2–3 puntos), y 83,4% vs. 83,1% es estadísticamente un empate, no un "ganador". La carrera en la cima es más cerrada de lo que la tabla sugiere.

Cursor 3.8 (18/06) trae Automatizaciones: a través de la habilidad /automate, puedes crear agentes siempre ejecutándose directamente desde la sesión de agente local.[9] Más un disparador de emoji de Slack, cinco nuevos disparadores de GitHub (comentarios de problemas, comentarios de revisión de PR, presentaciones de revisión de PR, actualizaciones de hilo de revisión, finalizaciones de ejecución de workflow) y una herramienta de computer-use para que agentes en la nube generen sus propios demos/artefactos.

Pruébalo Si tienes triajes de PR recurrentes o flujos de trabajo "en este emoji, inicia X": Automatizaciones es la primera vez que Cursor puede hacerlo sin código adhesivo desde la sesión. Bonificación: el Bugbot de Cursor ha revisado en ~90 segundos desde 10/06 (anteriormente ~5 min), encuentra ~10% más bugs con ~22% menos costo por ejecución.[9]

  • Aider aún v0.86.0; último push de repositorio 22/05. La tabla de benchmark de Aider ha estado sin actualizaciones durante meses — trata como "última verificación 22/05/2026, fecha de actualización de tabla desconocida," no como actual.
  • OpenCode (anomalyco) con 172k+ estrellas en GitHub, licencia MIT, 75+ proveedores — estable, sin nuevo lanzamiento.

  • Cline, Continue, Plandex, Devin, JetBrains AI: sin lanzamientos notables en la ventana de 7 días.

Es domingo, 21 de junio, y el breve de codificación de IA de hoy está moldeado no por lanzamientos, sino por dos decisiones regulatorias que sacan herramientas de los estantes.

La historia principal tiene nueve días y sigue siendo verdadera. Los dos modelos de codificación más fuertes de Anthropic, Fable cinco y Mythos cinco, han sido bloqueados desde el 12 de junio para todos los clientes. La razón es una orden de control de exportación del Departamento de Comercio de EE.UU. que prohíbe el acceso de ciudadanos extranjeros. Y como Anthropic no puede verificar la ciudadanía en la caja, ambos modelos tuvieron que estar fuera de línea completamente. Aún no hay una fecha de retorno confirmada. Si estabas ejecutando Claude Code con Fable cinco, estás cayendo al más débil Opus cuatro punto ocho.

Es por eso que duele tanto. Los mismos modelos están liderando los benchmarks. En Terminal-Bench dos punto uno, Codex con GPT cinco punto cinco está ligeramente adelante con ochenta y tres punto cuatro por ciento, con Claude Code usando Fable cinco justo atrás con ochenta y tres punto uno por ciento. En las tareas más difíciles, Fable cinco está incluso en el primer lugar. Entonces la mejor herramienta disponible está actualmente sin su mejor modelo. Aunque deberías leer estos números con escepticismo — la incertidumbre de medición es mayor que la diferencia en la cima.

El segundo apagón viene de Google. La línea de comandos Gemini ha estado muerta para cuentas de consumidores desde el 18 de junio. Su reemplazo se llama Antigravity y usa el comando agy, y a diferencia de la herramienta anterior, ya no es de código abierto. Si conectaste la antigua Gemini CLI en tus pipelines de construcción, debes arreglarlo esta semana, porque la integración de GitHub será completamente apagada el 17 de julio.

Más allá de eso, hubo lanzamientos reales. Cursor versión tres punto ocho trae automatizaciones — agentes que se ejecutan continuamente y se pueden activar directamente desde tu sesión, con disparadores de Slack y GitHub. Claude Code en versiones recientes bloquea comandos Git destructivos, que es una ganancia de seguridad real especialmente para ejecuciones desatendidas. Y la herramienta abierta OpenHands envía versión uno punto ocho con perfiles de LLM y delegación de subagentes — una alternativa abierta justo cuando dos opciones comerciales desaparecen.

Lo que hay que observar la próxima semana: cada palabra de Anthropic sobre Fable y Mythos regresando — y si aparece una primera presentación de Antigravity CLI en la tabla de clasificación de Terminal-Bench. Hasta entonces: planifica alrededor de disponibilidad, no características.
