Java · Kafka Streams API · AVRO · Microservices · Frankfurt / Remote

Stack: Kafka Streams, AVRO, AES, SOPS, Spring Boot, Microservices. Dauer: flexibel.

Kafka Streams API + AVRO: Kafka ist eine Pipeline für Event-Streams (denke „Förderband für Nachrichten"). Streams-API ist die Art, mit der man auf dem Förderband Logik laufen lässt — joinen, aggregieren, transformieren — ohne die Nachrichten erst in eine Datenbank zu schreiben. AVRO ist ein Daten-Format mit Schema-Pflicht: jede Nachricht trägt ein Schema, und ein zentrales Schema-Registry sorgt dafür, dass Producer und Consumer kompatibel bleiben. Wer das beherrscht, kann Real-time-Pipelines bauen, die nicht jeden Sprint zerbrechen.

Match: Kafka-Tiefe aus baseCON-Event-Streaming + Promonco-Microservices. AVRO als Format kennt man, sobald man einmal Schema-Registry-Probleme gelöst hat. Mittel-Hoch.
