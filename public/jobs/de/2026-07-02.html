twiceD-Kern: die Freelance-Wand

Der Primärmarkt liefert heute wieder eine dichte Reihe fast identischer Treffer — Java/Spring Boot + Angular, remote, DACH-Kunden. An der Spitze teclift (Fit 89): Fullstack Java Spring Boot 3 / Angular / Kafka, Köln, 100 % remote (nur alle ~6 Wochen 2–3 Tage onsite, bündelbar), 12 Monate ab 01.10., 90–115 €/h*. Das ist Daniels Brot-und-Butter-Stack 1:1 — deshalb der Höchstwert.[1]

Direkt dahinter drei 88er, alle Java/Spring mit langer Laufzeit: iSAX (Frankfurt, Java 21/Angular/AWS/Microservices, 48 Monate, 80 % remote), BridgingIT (Frankfurt, Spring Boot/K8s/Kafka, 45 Monate, 100 % remote) und RED SAP Solutions (Berlin, Java/Spring/Thymeleaf/PostgreSQL, Trust-Center/PKI-Nische, 24 Monate, Start Juli). Alle im 90–120 €/h*-Band, alle hoch-remote — vier solide Anbahnungs-Kandidaten.[2]

ANÜ (Arbeitnehmerüberlassung): Wie eine Leiharbeits-Konstruktion — der Vermittler stellt dich formal an und „verleiht" dich an den Endkunden, statt dass du als eigenständiger Freelancer direkt Rechnungen stellst. Für twiceD (eigene GmbH) unattraktiver als reines B2B, deshalb ziehen 1st Solution, Computer Futures, KMG u. a. im Score leicht runter.

Fit-Logik: stack_match (0–45) + location_remote (0–20) + rate_comp (0–15) + intrigue (0–10) + track_fit (0–10). Aufschlüsselung pro Listing im aufklappbaren „Fit"-Detail unter jeder Board-Zeile.

Markt-Signal & Filter

Nüchtern, ohne Coaching: Das Board konzentriert sich stark auf denselben Stack. Über alle 53 Listings zählen die Tags Java 43×, Spring Boot 36×, Angular 24×, Kafka 13×, Kubernetes 13× — Python/FastAPI taucht 7× auf (drei davon als eigener Sub-Cluster: Nemensis, percision, QUANTUM). Rund 39 der 53 sind DE-basiert mit hohem Remote-Anteil.

Zwei Struktur-Signale: die Premium-Nische bleibt schmal (5× Automotive/IoT), und die Employee-Spur bleibt eng (2 Treffer, beide am Gehalts-Floor). Sechs CH-Listings tragen Wohnsitz- oder ANÜ-Onsite-Auflagen und ranken deshalb niedrig; zwei Listings (Platri IT SÜ2, Orange Business CH-Wohnsitz) sind per hartem Gate auf 30 bzw. 40 gedrückt — bleiben im Board zur Transparenz, nie als Pitch.

Es ist Donnerstag, der zweite Juli. Das Board ist heute wieder gut gefüllt: dreiundfünfzig gescorte Einträge, davon achtundvierzig Freelance, zwei Angestellten-Positionen und drei aus der Kategorie „außerhalb der Box".

Ganz oben steht wie so oft der twiceD-Kernmarkt. Der beste Treffer kommt von teclift aus Köln mit einem Fit-Wert von neunundachtzig: Fullstack Java Spring Boot drei mit Angular und Kafka, hundert Prozent remote, zwölf Monate ab dem ersten Oktober. Das ist Daniels Brot-und-Butter-Stack, eins zu eins. Direkt dahinter drei Achtundachtziger von iSAX, BridgingIT und RED SAP Solutions — alle Java und Spring, alle mit langer Laufzeit und im Band um neunzig bis hundertzwanzig Euro pro Stunde.

Die Premium-Nische, in der Daniels Automotive-Referenzen und IoT-Skills wirklich zählen, ist heute dünn: nur fünf Listings, dafür mit höheren Raten. Am stärksten EnCo Software aus Hamburg mit einem Full-Stack-Projekt rund um autonomes Fahren und KI-Sicherheit, und HENNING eXperts im Ingolstädter Audi-Umfeld.

Die Angestellten-Spur bleibt eng. Nur zwei treffen den Filter: Aleph Alpha, das deutsche KI-Labor in Heidelberg, und Personio mit einer voll remoten Kotlin-Rolle. Beide haben denselben Vorbehalt — das Gehalt liegt am unteren Rand unter Daniels Hunderttausend-Euro-Schwelle.

Interessant außerhalb der Box: JetBrains sucht einen Principal Forward Deployed Engineer für KI-natives Entwickeln, remote in Europa — genau Daniels tägliches Feld. Anthropic und Helsing bleiben eher symbolisch drin, das eine US-primär, das andere onsite-lastig.

Ein Wort zur Datenqualität: nur zwei der dreiundfünfzig Einträge tragen ein echtes Posting-Datum, der Rest steht ehrlich als „Datum nicht angegeben". Und fünf Links zeigen auf eine generische Suchseite statt auf das konkrete Projekt — die sind vor einem Angebot manuell zu prüfen.

Bei den EU-Ausschreibungen läuft die Rahmenvereinbarung der Deutschen Flugsicherung weiter, Frist der dreizehnte Juli. Der Stack für das Software-Los passt, aber allein bietbar ist das nicht: das Personentage-Volumen, die Luftverkehrs-Referenzen und eine mögliche Sicherheitsüberprüfung übersteigen twiceDs Kapazität — hier bräuchte es einen Partner.

Unterm Strich: eine breite Java-Angular-Wand, eine schmale Premium-Nische und eine enge Employee-Spur. Morgen prüfen wir wieder, ob frische Herbst-Slots und neue Ausschreibungen dazukommen.
