Niche premium: autonomous driving [3]

EnCo Software (Fit 87) is looking for Java/Spring/EMF/React.js for safety-&-security engineering in autonomous driving, including AI failure analysis — 100% remote, 100–135 €/h*. It hits two CV strengths at once: Java fullstack and automotive-OEM references.

EMF (Eclipse Modeling Framework): a toolkit that automatically generates Java code from a formal model ("this is what a control unit looks like") — like a stencil that stamps out the same clean code every time instead of typing it by hand. In a safety context, this keeps the model and code guaranteed in sync, which matters for compliance evidence (ASPICE/ISO 26262).

Agentic-AI cluster: the market is now buying Claude Code proximity

Today's most interesting pattern isn't a single listing but a cluster: companies are posting roles that explicitly name agentic AI tooling — exactly Daniel's daily way of working.

Agentic AI / AI harness: instead of letting a language model just answer questions, you give it tools (read/write files, run code, run tests) and let it work through a task independently — like an intern who doesn't just give advice but does the work and checks it themselves. Claude Code and GitHub Copilot are such harnesses.

All three sit in the ninety-to-one-hundred-twenty-euro-per-hour band and match almost one-to-one with Daniel's core stack.

The most interesting role outside the top tier comes from EnCo Software: Java, Spring, and React for safety engineering in autonomous driving, including AI failure analysis. One hundred percent remote, one hundred to one hundred thirty-five euros an hour. That hits two of Daniel's strengths at once — Java fullstack and his automotive references.

What stands out today is a whole cluster of roles explicitly naming agentic AI tools — exactly the way of working with Claude Code that Daniel uses daily. ELCA in Zurich is looking for a senior Java engineer focused on agentic AI and names Claude Code and Copilot directly; the role was posted on June ninth and pays between one hundred forty and one hundred ninety thousand francs a year. Even Anthropic itself shows up on the board, with a role built around Claude Code. On the topic of hourly rates, there's fresh data. The Freelancer Compass twenty twenty-six, published on July second, puts the median at ninety-five euros an hour for IT freelancers in the DACH region, and ninety euros specifically for software and web development. The board's band of ninety to one hundred twenty euros sits cleanly at or slightly above the market — so the researched rates are set realistically. There was no fresh, named hiring freeze in the DACH region today, so none is mentioned. As a distant signal from the UK: IT contractor hiring grew again in May for the first time in over two years, driven by automation, AI, cloud, and cybersecurity.

On the public EU tenders via TED, today's filter matched one hundred sixty notices, but after close review only two genuine, currently open software contracts remain. The first comes from SPRIND, the Federal Agency for Disruptive Innovation: a feedback and analytics system built around the European identity wallet, deadline July twenty-fourth. The second is a traffic analytics platform for the state of Baden-Württemberg, deadline August fourth. Both match the stack, but both need a partner.

The reason is concrete: SPRIND requires a minimum annual revenue of six million euros in each of the last three years, an ISO certification, and three references from the public sector — all things a solo firm can't deliver alone. Baden-Württemberg even demands twenty full-time staff per year; that's the hard sticking point that twiceD can only meet as part of a consortium. Three other promising-sounding titles — from the Federal Network Agency, TU Berlin, and the University of Regensburg — I dropped, because they turned out to be already-awarded contracts with no open deadline.
