Agentic AI Cluster: the Market Keeps Buying Claude-Code Proximity

As in prior days, the most interesting pattern isn't a single listing but a cluster: companies are posting roles that explicitly name agentic AI tooling — exactly Daniel's daily way of working.

Agentic AI / AI harness: instead of letting a language model just answer questions, you give it tools (read/write files, run code, run tests) and let it work through a task on its own — like an intern who doesn't just give advice but does the work and checks it. Claude Code and GitHub Copilot are such harnesses.
