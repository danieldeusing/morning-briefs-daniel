Nicho premium · cluster embedded (quatro bullseyes)

O pico de terça de hoje entrega mais volume embedded que o meio típico da semana — quatro listagens, todas com link claramente defensável ao CV. Isso não é típico; geralmente uma ou duas listagens embedded de nicho por semana é a média. Posição de pricing do Daniel: €120–135/h pedido, €110/h piso.

U-Boot · Device tree · BSP: Imagine um sistema embarcado (placa de teste de semicondutor, central de controle de carro) como um mini-computador sem tela e sem BIOS. O U-Boot é a faísca inicial quando a energia é aplicada — o primeiro software que roda antes do sistema operacional propriamente dito inicializar. O device tree é a lista de peças que diz ao kernel Linux quais chips estão em quais pinos (em vez de compilar isso no kernel). O BSP (Board Support Package) é o stack customizado de U-Boot, device tree e módulos de kernel para exatamente aquela peça de hardware. Quem domina isso é o mecânico que coloca o carro para funcionar sem o manual — raro e caro.
