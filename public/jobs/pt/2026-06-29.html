Vários listings (DOS, Computer Futures, ITech) rodam neste modelo — para um diretor GmbH brasileiro, um framework tributário e de seguro social diferente de um contrato puro de trabalho, então verifique antes de assinar.

Premium vive no nicho IoT/Automotivo

Duas vagas se destacam da faixa commoditizada, e ambas merecem o marcador hot: Keil GmbH (Fit 86) busca Java/Spring Boot para telemetria/IoT com MQTT, remoto DACH, taxa pesquisada 100–130 €/h*.[4] E Computer Futures (Fit 83) tem um papel Java no espaço e-mobilidade/automotivo, 100% remoto, também 100–130 €/h*.[5]

Por que esses são os únicos dois entries onde um markup é defensável: integração de protocolo IoT e software automotivo são raramente anunciados na região DACH, e Daniel tem referências reais no espaço OEM — a plataforma de vendas VW (VU3, Java/Spring/AWS, desde 2021) e o driver Linux GigE Vision para Mercedes-Benz & Bosch (40 Gbit/s de throughput). Isso não é "consigo aprender", é "entreguei". Exatamente lá o prêmio de 10–30 €/h acima da taxa Java padrão se mantém.

MQTT: Um protocolo de mensageria extremamente enxuto para dispositivos com pouca energia e banda — pense em um rádio que envia códigos curtos em vez de frases completas. Em telemetria (telemetria de veículos) e IoT, MQTT envia mensagens de sensor e status para um broker central, que as reencaminha para sistemas de análise.

Veredicto: Negociação de taxa recomendada aqui, não defesa de piso. Em ambas, traga as referências OEM para a faixa superior (130 €/h) — o nicho justifica, o mercado Java padrão não.
