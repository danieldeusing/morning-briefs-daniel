Premium de nicho: direção autônoma

EnCo Software (Fit 87) busca Java/Spring/EMF/React.js para engenharia de safety & security em direção autônoma, incluindo análise de falhas com IA — 100% remoto, 100–135 €/h*. Isso acerta duas forças do currículo ao mesmo tempo: fullstack Java e referências em OEMs automotivas.

EMF (Eclipse Modeling Framework): um conjunto de ferramentas que gera código Java automaticamente a partir de um modelo formal ("é assim que uma unidade de controle se parece") — como um molde que sempre estampa o mesmo código limpo, em vez de digitá-lo à mão. Em contextos de safety, isso mantém modelo e código garantidamente sincronizados, o que importa para evidências de conformidade (ASPICE/ISO 26262).

Completando o board: Computer Futures e-Mobility/Automotive (Java/Spring Boot/MQTT/CAN, 100–130 €/h*, Fit 82), HENNING eXperts Automotive Ingolstadt (Java/Groovy/Spring Boot/Angular, ambiente Audi, Fit 86) e VARIUS IT Smart Energy (Java/Kafka/K8s, 100–130 €/h*, Fit 85). Cluster de IA agêntica: o mercado continua comprando proximidade com Claude Code

Como nos dias anteriores, o padrão mais interessante não é uma vaga isolada, mas um cluster: empresas publicam papéis que mencionam explicitamente ferramental de IA agêntica — exatamente o jeito de trabalhar de Daniel no dia a dia.

IA agêntica / AI harness: em vez de deixar um modelo de linguagem apenas responder perguntas, você dá a ele ferramentas (ler/escrever arquivos, executar código, rodar testes) e deixa que ele conduza uma tarefa sozinho — como um estagiário que não só dá conselhos, mas faz o trabalho e confere. Claude Code e GitHub Copilot são esses harnesses.
