Vocabulary: Databases — to query, sincronizar, der Container

Tuesday, July 1 — Weekly theme Software & Technology, today Part VI: Databases, APIs & Cloud Infrastructure. The vocabulary for querying data (consultar), synchronizing services, monitoring a system, migrating a database to the cloud, and scaling everything. 32 new entries, 12 verbs with full conjugation — including the notorious Spanish trap yo sincronicé (z→c) — plus 3 reviews from last Tuesday's AI-tools session (automatisieren, konfigurieren, debuggen). Grammar focus: separable (ich frage ab) vs. inseparable (ich überwache), and why -ieren verbs never take a ge- prefix.

  • FALSE FRIEND DE Anfrage ≠ EN inquiry — in a tech context, an Anfrage is a request (HTTP request, API call), not merely a formal inquiry. PT a requisição / a solicitação, ES la solicitud / la petición. EN request is the best match.
  • FALSE FRIEND PT/ES implementar ≠ DE implantieren — to implement something in software is PT/ES implementar. DE implantieren is narrower (surgery, implanting hardware). For software in all three languages: implementieren / implementar.
  • CONJUGATION ES sincronizar → yo sincronicé — Spelling trap: -zar verbs write c instead of z before e. Preterite 1st person sincronicé (never *sincronizé), full subjunctive sincronice. PT has no such trap: eu sincronizei.
  • CONJUGATION DE abfragen → ich frage ab — separable verb: the prefix ab- splits off in the main clause (ich frage die Daten ab). Past participle abgefragt — the ge- sits between prefix and stem.
  • CONJUGATION DE überwachen → ich überwache — inseparable verb: über- stays fixed to the stem. Past participle überwacht — no ge-, because inseparable. Compare to the separable abfragen → abgefragt.
  • CONJUGATION DE -ieren → no "ge-" — synchronisieren, migrieren, implementieren, skalieren, replizieren, indexieren, archivieren all form the past participle without ge-: synchronisiert (not *gesynchronisiert), migriert, skaliert.
  • CONJUGATION Anglicisms: cachen & deployen — both weak and regular: ich cache, du cachst, habe gecacht; ich deploye, du deployst, habe deployt/deployed. No fixed standard — common in everyday use, participle has variant spellings.
  • GENDER die Datenbank (f), der Datensatz (m), das Schema (n) — three genders side by side. PT o banco de dados (m), o registro (m), o esquema (m); ES la base de datos (f), el registro (m), el esquema (m). Never guess the gender.
  • BR-PT banco de dados vs. base de dados — "database" in Brazilian PT is o banco de dados (universal in Brazil); European PT says a base de dados. ES always la base de datos. Also: o contêiner (BR) vs. o contentor (PT-EU).
  • BR-PT a nuvem vs. a cloud — "the cloud" in formal Brazilian PT is a nuvem; a cloud is the colloquial anglicism, widely used in Brazil's tech scene. Both are understood; prefer a nuvem in written text.
  • IDIOM Die Datenbank ist down. = PT O banco de dados está fora do ar. = ES La base de datos está caída. — "down / fora do ar / caída" is the idiomatic way to say a system is unreachable.
  • PHRASE etwas in der Cloud hosten = PT hospedar algo na nuvem = ES alojar algo en la nube. "Das System skaliert automatisch" = O sistema escala automaticamente = El sistema escala automáticamente.

Database Verbs: to query, synchronize, migrate

12 verbs with full conjugation + reviews · DE · EN · PT-BR · ES-MX

Separable vs. inseparable (DE): some German verbs have a prefix that detaches in the sentence — like a trailer being uncoupled. abfragen is separable: ich frage die Daten ab. überwachen is inseparable: the über- always stays on the stem — ich überwache das System. Rule of thumb for the participle: separable gets ge- in the middle (abgefragt), inseparable gets no ge- at all (überwacht). -ieren verbs (borrowed from Latin: synchronisieren, migrieren, skalieren) also drop the ge-: synchronisiert, never *gesynchronisiert. -zar trap (ES): in Spanish a z before an e becomes c — so for sincronizar the preterite is yo sincronicé and the entire subjunctive is sincronice. This keeps the sound consistent.

Grammar of the Day Separable vs. inseparable — and where the "ge-" lands. With a separable verb, the prefix splits off in the main clause (ich frage die Daten ab), and in the past participle the ge- slides between prefix and stem: abgefragt. With an inseparable verb the prefix stays fixed on the stem (ich überwache das System) and the participle gets no ge- at all: überwacht. Inseparable prefixes are be-, ge-, er-, ver-, zer-, ent-, emp-, miss- and usually also über-, unter-, um-, durch- (depending on meaning). And -ieren verbs (synchronisieren, migrieren, skalieren) also drop the ge-: synchronisiert, never *gesynchronisiert.
Idiom "etwas in trockenen Tüchern haben". Literally: something is dry, meaning finished and safely packed up. Figuratively: a matter is done and dusted, nothing can go wrong anymore — e.g. "Die Migration ist in trockenen Tüchern, wir können deployen." EN to have something in the bag / all wrapped up · PT estar com tudo certo / no papo · ES tener algo amarrado / cerrado.

It's Tuesday, July 1. This week's theme is Software and Technology, and today's focus is databases, APIs, and cloud infrastructure — querying data, synchronizing services, monitoring a system, and migrating a database to the cloud.

The centerpiece today is a German grammar contrast: separable versus inseparable. The verb abfragen is separable — the prefix ab splits off: ich frage die Daten ab. The past participle is abgefragt, with ge tucked between the prefix and stem. The verb überwachen, by contrast, is inseparable — über stays glued to the stem: ich überwache das System. And the participle drops the ge entirely: überwacht.

Layered on top is the rule for the many -ieren verbs. Synchronisieren, migrieren, implementieren, skalieren, replizieren, and indexieren all form their past participle without ge. It's synchronisiert, never gesynchronisiert; migriert, never gemigriert.

The most important conjugation trap today is in Spanish. The verb sincronizar ends in -zar, and before an e the z becomes c. In the preterite, first person, it is therefore yo sincronicé — with c, never sincronizé. The entire subjunctive is also spelled with c: sincronice. Portuguese has no such trap: it's simply eu sincronizei.

Two false friends are worth flagging. In a tech context, Anfrage means request — an HTTP request or API call — not merely a polite inquiry. And implementieren in the software sense is implementar in both Portuguese and Spanish; the German implantieren is narrower and refers to surgical implantation.

On the noun side, it pays to note gender and the Brazilian variant. "Database" in Brazilian Portuguese is o banco de dados; European Portuguese says a base de dados. And "the cloud" is formally a nuvem, colloquially a cloud. The key practice phrase: the database is down — Portuguese está fora do ar, Spanish está caída.

Three verbs return as a review from the AI-tools Tuesday: automatisieren, konfigurieren, and debuggen. All three fit the infrastructure theme perfectly.
